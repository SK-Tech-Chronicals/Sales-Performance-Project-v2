# 📊 Sales Dashboard (Excel Project)

## 🔹 Project Overview

This project is an interactive **Sales Dashboard** created using Microsoft Excel.
It helps in analyzing sales performance and tracking key metrics based on orders, revenue, and profit.

---

## 📂 Dataset

The dataset contains sales-related information such as:

* Order Date 📅
* Product / Category 📦
* Sales Amount 💰
* Profit 📈
* Region 🌍

---

## 🔹 Features

✔️ Tracks overall sales performance
✔️ Sales vs Profit comparison
✔️ Region-wise and category-wise analysis
✔️ Interactive filters using Slicers
✔️ Pivot Table-based analysis
✔️ Visual charts for better understanding

---

## 🔹 Dashboard Insights

* Total sales and profit overview
* Monthly sales trend analysis
* Top-performing categories/products
* Region-wise performance comparison
* Identifies high and low performing areas

---

## 🔹 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning & Formatting

---

## 🔹 File Structure

* Dashboard Sheet → Final visual dashboard
* Raw Data Sheet → Original dataset
* Processed Data Sheet → Cleaned and structured data

---

## 🔹 Key Learnings

* Data cleaning and transformation
* Creating Pivot Tables for business insights
* Building interactive dashboards
* Sales data analysis and visualization
